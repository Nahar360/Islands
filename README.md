'Islands' project developed with `SFML`, `imgui` and `imgui-sfml`.

# Building and running

```sh
cd islands
cmake -S . -B build
cmake --build build
./build/src/islands
```

Alternatively, you can build and run 'Islands' using Visual Studio Code with the launch configuration provided.
